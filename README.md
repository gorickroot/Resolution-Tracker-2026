# 🎯 2026 Resolution Tracker

A lightweight, gamified web app to track your daily resolutions and build streaks throughout 2026.

🌐 **Live:** [resolution-tracker-2026.netlify.app](https://resolution-tracker-2026.netlify.app)

---

## ✨ Features

- 🔐 Sign up & login — no backend, works fully in the browser
- ➕ Add, edit & delete resolutions
- ✅ Mark resolutions as done each day
- 🔥 Streak counter — tracks consecutive days per resolution
- 📊 Stats dashboard — total goals, done today, best streak
- 📅 2026 heatmap — visual calendar of your full year progress
- 🎮 Dark pixel-style gamified UI

---

## 🧱 Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | HTML + CSS + Vanilla JS |
| Storage | Browser localStorage |
| Hosting | Netlify (free) |

Zero dependencies. Zero frameworks. Single `index.html` file.

---

## 🚀 Run Locally

Just open `index.html` in any browser — no server, no setup needed.

---

## 📦 Deploy Your Own

1. Fork this repo
2. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import from GitHub**
3. Select this repo, leave all settings default
4. Click **Deploy** — live in seconds ✅
